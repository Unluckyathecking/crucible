# Work order — async-job-poll-sdk (`async-job-poll-sdk`)

**Lane:** `10xworker:job` · **Module:** `async-job-poll-sdk` · **Seeded:** 2026-07-14 (cycle 2)

## Context

PRs #175–#184 built the entire **server** half of durable async jobs: enqueue → `202 {job_id}`,
`GET /v1/jobs/{id}` polling, `GET /v1/jobs` list, retry/dead-letter, completion webhooks, and the
retention reaper. The polyglot consumer-SDK trio (Go/TS/Python) was *just* completed in #183 — but
each SDK exposes only the generated `GetJob`/`ListJobs` (`clients/go/client.go:316`,
`clients/typescript/src/client.ts:199`, `clients/python/crucible_client/client.py:306`). **No SDK
has a polling/wait helper** (`grep -riE 'waitfor|wait_for|poll|backoff' clients/` → zero hits). Every
customer of every clone that opts a route into `AsyncRoutes` must hand-roll a poll loop: backoff,
timeout, terminal-status detection, failed-job error mapping. This is the missing client half of the
contract the framework just finished, and it compounds across three languages × every async-enabled
clone.

**Clean seam (low architectural risk):** the SDKs already ship hand-written companions that the
generator explicitly does not own — `clients/go/webhook.go`, `clients/typescript/src/webhook.ts`,
`clients/python/crucible_client/webhook.py` (banner: "Hand-maintained — NOT written by
scripts/gen-clients.sh"). A `jobs` companion slots into that exact seam, so it coexists with the
`client-sdk-drift.yml` `git diff --exit-code clients/` gate just as the `webhook.*` companions
already do.

## Rationale

The server-side async subsystem is complete through the retention reaper (#184), and the SDK trio
was completed in #183, yet not one SDK offers a wait/poll helper — so every integrator hand-rolls
backoff + terminal detection + failed-job mapping across three languages and every async-enabled
clone. Adding a hand-written `jobs.*` companion beside the existing `webhook.*` companions is the
reusable client half of the contract, lands in a generator-exempt seam already blessed by the drift
gate, and stays fully within `clients/**` — disjoint from #167 (`ee/**`, `gateway/internal/license/**`,
`gateway/cmd/licensegen/**`, `dashboard/lib/license.ts`, `dashboard/app/page.tsx`) and #168
(`.github/**`). ~600–900 LOC, well under the 10k cap.
